# Getting Started with Lumi

Welcome to **Lumi**, the minimalist, responsive assistant that understands, thinks, and responds directly from your screen. Built by developers, for developers, Lumi integrates directly into your Windows taskbar.

## What is Lumi?

Lumi is an open-source AI assistant designed for seamless integration into your daily workflow. It lives quietly on your screen and provides instant access to system controls, file management, web scraping, and developer tools.

### Key Features
- **Seamless Integration:** Native taskbar localization.
- **Ambient Feedback:** Glowing waveforms and particle fields.
- **Minimal Footprint:** Takes up less than 1/10th of your screen coverage.
- **Developer-Ready:** Fork, build, and customize to your heart's content.
